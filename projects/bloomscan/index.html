BloomScan

Identify flowers instantly from a single photo.

GitHub Live Demo

Overview

BloomScan is a visual identification tool designed to help users recognize flowers and access essential care information from a single photo. The challenge was presenting plant data in a way that feels immediate and easy to understand, rather than technical or overwhelming. The solution was a streamlined interface that accepts an image input and returns a structured profile including species identification, care guidance, and safety information.

Key Features

  • Photo-Based Identification

    Uploads an image and returns likely flower identification quickly.

  • Care Guidance Summary

    Displays concise care instructions for watering, sunlight, and handling.

  • Structured Result Cards

    Presents identification and safety details in a clean, readable format.

  • Fast Interaction Flow

    Minimizes steps between image upload and actionable plant insights.
