AutoWorth

Evaluate car deals in seconds.

GitHub Live Demo

Overview

AutoWorth is a decision-support tool that helps users evaluate vehicle listings by identifying pricing inconsistencies and potential risks. The challenge was interpreting unstructured listing data and presenting it in a way that supports quick decision-making. The solution involved creating a simplified analysis system that compares key indicators and provides clear recommendations such as buy, negotiate, or avoid.

Key Features

  • Value Score System

    Summarizes listing quality through a single readable score.

  • Buy / Negotiate / Avoid

    Provides clear recommendation states for faster decisions.

  • Price vs Market Insight

    Compares listing values against typical market expectations.

  • Mileage Analysis

    Assesses usage patterns relative to listing age and value.

  • Risk Indicators

    Flags potential red flags before purchase commitment.
